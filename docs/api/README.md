# API Documentation

This directory contains documentation for the FoodDeli API.

## Overview

FoodDeli uses a combination of Firebase services and custom APIs to provide functionality for the food delivery application.

## Firebase APIs

### Firestore Database
- User profiles and authentication
- Restaurant information and menus
- Order management and tracking
- Delivery information

### Firebase Authentication
- User registration and login
- Social login integration
- Password reset functionality

### Firebase Cloud Messaging
- Push notifications for order updates
- Delivery status notifications
- Promotional messages

## Custom APIs

### Order Management API
- Order creation and processing
- Payment integration
- Status updates

### Restaurant API
- Menu management
- Availability updates
- Rating and review system

### User API
- Profile management
- Address book
- Payment methods

## Authentication

API authentication is handled through Firebase Authentication tokens.

## Rate Limiting

API rate limits are managed by Firebase services.
