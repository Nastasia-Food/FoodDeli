## How to Become a Supporter

Your support helps ensure the continued development and maintenance of FoodDeli. See [FUNDING.md](FUNDING.md) for details on how to contribute.

### Benefits by Tier

#### Bronze ($10+/month)
- Name listed in this document
- Recognition in release notes
- Early access to releases

#### Silver ($50+/month)
- All Bronze benefits
- Priority support
- Exclusive community access

#### Gold ($100+/month)
- All Silver benefits
- Special badge on GitHub profile
- Voting rights on roadmap priorities

#### Corporate ($500+/month)
- All Gold benefits
- Logo placement in README
- Dedicated account management
- Custom development services

## Transparency

We update this list monthly to reflect current supporters. If you prefer not to be listed, please indicate this when making your contribution.

## Recognition

Supporters are recognized in the following ways:
- Listing in this document
- Mention in release notes
- Special badges for GitHub Sponsors
- Exclusive access to supporter community

## Questions

If you have questions about sponsorship or would like to be added to this list, please contact support@rechain.network.

Thank you to all our supporters for making FoodDeli possible!
